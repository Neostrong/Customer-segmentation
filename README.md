Customer Segmentation Project: This project segments customers based on demographics and purchase behavior and predicts campaign response using machine learning. The dataset includes Age, Income, Total Spend, Recency, Web/Store Purchases, Marital Status, and Education, with the target being Response. Steps include data cleaning and feature engineering, clustering with KMeans into four segments (High-value loyal, Low-spend browsers, Digital-first, Price-sensitive families), and predictive modeling with Random Forest and Decision Tree. Visualizations include cluster scatter plots and feature importance charts. To run: install pandas, numpy, scikit-learn, matplotlib, seaborn
